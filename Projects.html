ExchangeBrowse

This C# based application was made during my internship at Commvault Systems in Tinton Falls, NJ. The application grabs archived mail data from an Appache Solr index, and displays the data in a browse view, with tabs for subject, from, to, cc, and some other archived fields. In addition, the application uses an authentication which allows authenticated users to modify the data and send the data back to the Solr index. The application support's Solr's atomic update feature, as well as support for batch updates. The project took 2 full summers, working 9-5 every weekday, to complete.


Command Genie

This C# based application has 2 parts: the application, and the addin required for the application to run. The application is a Windows Daemon application that scans for incoming mail to a configured email address. If the subject of the incoming mail is "EXECUTE COMMAND", the body of the mail is then scanned. The body of the mail can contain commands to execute on the machine's command line. The command(s) are executed, and the response is sent back in an email response to the user that sent the command email. If the response is longer than a configurable length, the response is written to a text file and sent back to the user as an attachment. The addin is a daemon that actually looks for the email, and changes a registry value when it finds a command email. This registry change then triggers the actual application.

Worked with a friend on a multiplayer third person shooter for our Virtual Reality and Game Design course. The project was made using Unity Engine and C# scripting. My contributions included level design, networking, UI / HUD, and player movement / animations. All of the assets used in the project were purchased or downloaded legally. We are still adding things to this project, as well.


CS:GO Stat Graphing Tool

This C# based Windows Application that uses Steam's API and the graphing API OxyPlot to display a CSGO player's ingame weapon usage stats, given their steam ID. There is also an option to compare a user's weapon stats with a friend's weapons stats.
